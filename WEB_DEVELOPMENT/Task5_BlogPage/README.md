# My Blog Page

## Overview

My Blog Page is a responsive blogging website built using HTML, CSS, and JavaScript. It showcases multiple blog posts with summaries, detailed articles, and an interactive "Read More" feature.

## Features

- Responsive Design
- Modern UI with Gradient Backgrounds
- Multiple Blog Posts
- Author and Date Information
- Post Summaries
- Expandable "Read More" Sections
- Smooth Hover Effects
- Mobile-Friendly Layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts (Poppins)

## How to Run

1. Download the project files.
2. Save the code as `index.html`.
3. Open the file in any modern browser.

No installation required.
