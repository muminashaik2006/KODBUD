# Tech Fest 2026

## Overview

Tech Fest 2026 is a responsive event management website built using HTML, CSS, Bootstrap 5, and JavaScript. The website showcases technology-related events including AI Workshop, Hackathon, and Robotics Expo with interactive modal popups containing detailed information about each event.

## Features

* Responsive Bootstrap 5 design
* Attractive hero section with gradient background
* Event cards with images
* Interactive modal popups
* Mobile-friendly navigation bar
* Event details including:

  * Authorization information
  * Importance of the event
  * Future career opportunities
* Modern UI with custom styling

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5.3.3
* JavaScript (Bootstrap Bundle)

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.

No server setup is required.
