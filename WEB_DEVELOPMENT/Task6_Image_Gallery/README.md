# Responsive Image Gallery

## Overview

Responsive Image Gallery is a simple and modern web project that displays a collection of beautiful images using a responsive layout. The gallery automatically adjusts its structure based on the screen size, providing an optimal viewing experience on desktops, tablets, and mobile devices.

## Features

- Responsive Gallery Layout
- Mobile-Friendly Design
- High-Quality Unsplash Images
- Clean and Minimal User Interface
- Easy to Customize
- Fast Loading Design

## Technologies Used

- HTML5
- CSS3

## Responsive Design

### Desktop
- Displays images in a multi-column grid layout.

### Tablet
- Automatically adjusts the number of columns.

### Mobile
- Displays images in a single-column layout for better visibility.

## How to Run

1. Download the project files.
2. Save the HTML file as `index.html`.
3. Save the CSS file as `style.css`.
4. Open `index.html` in any web browser.

No installation is required.
