
# Advanced Google Homepage Clone

## Project Overview

This project is a responsive clone of the Google homepage built using HTML5 and CSS3. It recreates Google's modern dark-themed interface with animated logo effects, responsive search bar, navigation links, and footer sections.

## Features

- Responsive Design
- Dark Theme Interface
- Animated Google Logo
- Search Bar UI
- Gmail and Images Navigation
- Language Selection Links
- Footer Navigation
- Mobile-Friendly Layout
- Pure HTML and CSS

## Technologies Used

- HTML5
- CSS3

The website adapts automatically for:

- Desktop Screens

## Animation

The Google logo includes a floating animation created using CSS keyframes.

## How to Run

1. Download the project files.
2. Save the code as `index.html`.
3. Open the file in any modern web browser.

No installation or dependencies are required.
