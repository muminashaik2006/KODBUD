# EcoRide - Responsive Landing Page

## Project Overview

EcoRide is a modern and responsive landing page for a fictional electric bike startup. The website promotes eco-friendly transportation by showcasing product features, customer testimonials, and a test ride booking experience.

## Features

- Fully Responsive Design
- Modern Hero Section
- Feature Highlights Section
- Customer Testimonials
- Interactive Test Ride Popup
- Smooth Navigation
- Mobile-Friendly Layout
- Pure HTML, CSS, and JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Sections Included

### Header
- EcoRide logo
- Navigation menu

### Hero Section
- Product introduction
- Call-to-action button
- Electric bike image

### Features Section
- Long Battery Life
- Fast Charging
- Eco-Friendly Design
- Smart App Control

### Testimonials Section
- Customer reviews and feedback

### Footer
- Contact information
- Copyright details

### Test Ride Popup
- Opens when user clicks "Book a Test Ride"
- Displays test ride details
- Schedule confirmation alert

## How to Run

1. Download the project files.
2. Save the HTML file as `index.html`.
3. Open `index.html` in any web browser.
