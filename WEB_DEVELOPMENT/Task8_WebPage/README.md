# Animated Nature Gallery

## Overview

Animated Nature Gallery is an interactive front-end web project that showcases different natural environments through visually appealing cards and animated scenes. Users can explore Mountains, Desert, Ocean, and Forest environments by clicking on gallery cards, which open immersive animated views.

## Features

* Responsive gallery layout
* Attractive animated page title using Animate.css
* Interactive image cards
* Modal-based scene viewer
* Multiple nature-themed environments:

  * Snowy Mountains
  * Desert Storm
  * Ocean Waves
  * Green Forest
* Custom animations and visual effects
* User-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Animate.css

## Nature Scenes

### Mountains

Displays a snowy mountain environment with animated snowfall effects.

### Desert

Presents a desert storm experience with moving sand animations.

### Ocean

Shows realistic ocean wave animations.

### Forest

Creates a forest environment with animated falling leaves.

## How to Run

1. Download or clone the project files.
2. Ensure all files are in the same directory:

   * index.html
   * style.css
   * script.js
3. Open `index.html` in any modern web browser.

No installation or server setup is required.
