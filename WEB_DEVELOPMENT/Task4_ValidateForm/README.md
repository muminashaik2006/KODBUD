# Job Application Form

## Overview

This project is a responsive Job Application Form built using HTML, CSS, and JavaScript. It allows users to enter their personal information and validates the form before submission.

## Features

- Responsive Design
- Elegant Purple Gradient Theme
- Italic Typography using Playfair Display
- Form Validation using JavaScript
- Error Messages for Invalid Inputs
- Success Message after Submission
- Mobile Friendly Layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts (Playfair Display)

## Form Fields

- Full Name
- Email Address
- Phone Number
- Message

## Validation Rules

### Name
- Cannot be empty

### Email
- Cannot be empty
- Must follow valid email format

### Phone Number
- Cannot be empty
- Must contain exactly 10 digits

### Message
- Cannot be empty

## How to Run

1. Download the project files.
2. Save the code as `index.html`.
3. Open the file in any modern web browser.

No installation required.
