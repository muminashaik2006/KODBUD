# Personal Portfolio Website

## Overview

This is a responsive personal portfolio website developed using HTML and CSS. The portfolio showcases personal information, education, skills, projects, and contact details in a clean and modern design.

## Features

* Responsive navigation bar
* Hero section with profile image
* About Me section
* Education details
* Skills showcase
* Projects display
* Contact form
* Modern and attractive UI

## Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
